## What is sptfy.in?

sptfy.in is a **simple Spotify link shortener** that transforms your long Spotify URLs into clean, shareable links.

Paste your Spotify track, album, or playlist URL → add a custom slug (optional) → done. No ads, no nonsense—just short links that actually work.

## What You Can Shorten

sptfy.in supports all major Spotify content types:

- Tracks (`/track/...`)
- Albums (`/album/...`)
- Playlists (`/playlist/...`)
- Artists (`/artist/...`)
- Shows/Podcasts (`/show/...`)
- Episodes (`/episode/...`)

<Note>
  **spotify.link support**: sptfy.in automatically expands `spotify.link` short URLs before processing them.
</Note>

## Tech Stack

Built with modern, reliable technologies:

- **Frontend**: SvelteKit + Svelte 5 (runes)
- **Backend**: PocketBase (Go-based backend as a service)
- **UI**: Tailwind CSS + shadcn-svelte components
- **Validation**: Cloudflare Turnstile (bot protection)
- **Hosting**: Cloudflare Pages (frontend) + Docker VPS (backend)
